# basic-greeter

## greet-helloworld

A very simple static hello world greeter.
This skill should ONLY be called when the user say Hi.

## greet-helloname

A very simple greeter that include the name of the user if it was introduced earlier in the context.

## greet-helloname-withdatetime

A very simple greeter that include the name of the user if 
it was introduced earlier in the context,
using the following format: "YYYY-MM-DD: Hello, <actualname>"
